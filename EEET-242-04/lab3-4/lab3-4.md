## Lab 3

### Resistive Circuit 1

| Measurement | Value          |
| ----------- | -------------- |
| $V_p$       | $120.17V$      |
| $V_s$       | $58.7V$        |
| $I_p$       | $0.0962A$      |
| $I_s$       | $0.1425A$      |
| $P_p$       | $11.30W$       |
| $P_s$       | $8.48W$        |
| $S_p$       | $11.56VA$      |
| $S_s$       | $8.36VA$       |
| $a$         | $\approx2.047$ |

$$
V_p,V_s,I_p,I_s,P_p,P_s\ measured\\
S_p=V_p\times I_p=120.17V\times0.0962A=11.56VA\\
S_s=V_s\times I_s=58.7V\times0.1425A=8.36VA\\
a=\frac{V_p}{V_s}=\frac{120.17V}{58.7V}\approx2.047
$$
